Class RechnungspositionFlat

Diese Klasse ist der Rechnungsposition nachempfunden und dieser ähnlich. Jedoch enthält sie weniger Daten und v.a. in einem flachen Format, das mit den beschränkten SAP Bordmitteln leichter als Entitätstyp für die Verarbeitung in einer OData-Schnittstelle abgebildet werden kann.
